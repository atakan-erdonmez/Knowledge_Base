Below are the Common Linux default log files name and their usage:

- `/var/log/message` тАУ Where whole system logs or current activity logs are available.
    
- `/var/log/auth.log` тАУ Authentication logs.
    
- `/var/log/kern.log` тАУ Kernel logs.
    
- `/var/log/cron.log` тАУ Crond logs (cron job).
    
- `/var/log/maillog` тАУ Mail server logs.
    
- `/var/log/boot.log` тАУ System boot log.
	
- `/var/log/mysqld.log` тАУ MySQL database server log file.
    
- `/var/log/secure` тАУ Authentication log.
    
- `/var/log/utmp` or `/var/log/wtmp` тАУ Login records file.
    
- `/var/log/yum.log` тАУ Yum log files.
